# Operating system support

This module has been tested on

* Debian 7 and 8
* Ubuntu 12.04 and 14.04
* CentOS 6
* FreeBSD 10

Any *NIX-style operating system should work out of the box or with small 
modifications.

For details see [params.pp](manifests/params.pp).
